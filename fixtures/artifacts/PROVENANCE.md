# Artifact fixture provenance

The digest fixtures in this directory were generated by Sibylla from the
repository's original metadata-only deck and fictional reading fixtures. They
contain no artwork, guidebook text, personal reading data, secrets, or model
weights.

The files pin the SHA-256 content IDs of artifact-envelope schema version 1.
Changing one requires an intentional compatibility review of the canonical
serialization contract and the corresponding test fixture.
